1. 상수란 무엇이며 변수와의 차이는?
- 상수란 변수에 값을 딱 한번만 할당할 수 있는 수이다. 변수와 다른점은 한번 할당된 값은 변경이 
   불가능 하다는 것이다.

2. 초기화란?
- 값을 첫번째로 할당하는 행위이다.

3. 리터럴이란?
- 자료형을 기반으로 표현되는 상수이다.

4.아래의 내용이 에러가 나는 이유와 수정 방법은?

java
System.out.println(3147483647 + 3147483648); //에러남
>>> 정수는 무조건 int형으로 인식하기로 되어있다. 그러나 해당 숫자는 int의 범위를 넘어간 것이므로
      해당 숫자들 뒤에 l또는 L을 붙여 int보다 큰 long형으로 컴파일러한테 요청한다.

6.아래의 에러가 나는 이유를 설명하고 수정 하시오.

int a = 3;
int b = 4;
	
double result = a / b;

System.out.println(result);
>>> 정수형으로 연산을 해서 소수점 단위를 버린다. 이에 대한 결과는 0이 나오지만 앞의 double로 인해
      0.0의 결과가 나온다. 오류수정은 a또는 b 자료형 앞에 (double)을 넣어서 명시적 형 변환을 시켜준다.

11. && 와 || 설명하시오.
- &&는 and 연산자이다. 앞 뒤로 모두 true값이 나와야지만 결과값이 true가 나오고,
  ||는 or 연산자이다. 둘 중 하나만 true값이 나와도 결과값은 true가 나온다.

public class Hello {

		int num = 1;		
		System.out.println(ch);

	}
}

출력결과
49
1
*/
>>>첫번째 변수 ch는 작은 따옴표로 인한 문자 '1'에 해당하는 유니코드값인 49가 출력되고,
